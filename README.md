# Öğrenci Portfolyo Web Sayfası

Bu proje, HTML ve CSS kullanılarak hazırlanmış kişisel portfolyo web sayfasıdır.

## Kullanılan Teknolojiler

- HTML
- CSS
- Font Awesome

## Bölümler

- Anasayfa
- Hakkımda
- Yetenekler
- İletişim

## Amaç

Bu proje, web tasarımı ve kodlama öğrenme sürecimde yaptığım ilk portfolyo çalışmalarından biridir.
